🇪🇹 Ge'ezCode ProGe'ezCode Pro is an innovative IDE (Integrated Development Environment) designed to empower Ethiopian developers by allowing them to write logic using the Ge'ez script and Amharic syntax.🚀 Vision"ባህልና ቴክኖሎጂ ለታላቅቷ ኢትዮጵያ" (Culture and technology for a Great Ethiopia)Our goal is to lower the barrier to entry for coding in Ethiopia and promote digital sovereignty.✨ Key FeaturesGe'ez Syntax: Code using familiar Amharic keywords like ተለዋዋጭ (Variable) and አትም (Print).Multi-Language Transpiler: Automatically converts Ge'ezCode into executable JavaScript and Python.AI-Powered Code Generation: Integrated with Google Gemini AI to generate code from natural language prompts.Hardware & Visual Simulation: Built-in canvas for visual output and simulated hardware controls (e.g., Irrigation Pump simulation).Responsive Terminal: A localized console output for debugging.🛠️ Tech StackFrontend: HTML5, Tailwind CSSLogic: Vanilla JavaScript (ES6+)AI Integration: Google Generative AI (Gemini API)Deployment: GitHub Pages📖 How to UseOpen the Live Demo.Select a lesson from the sidebar to see example code.Use the AI Generator to request code in Amharic (e.g., "ክበብ ሳልልኝ").Click አስጀምር to run the code and see results in the Console and Canvas.
